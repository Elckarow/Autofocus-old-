Feature: DropShadow

Adds a drop shadow effect to the character's sprites.


allowed arguments:
xoffset              defaults to 3
yoffset              defaults to 3
offset               optionnal, will override both xoffset and yoffset if passed
blur                 defaults to 2.5


If the version of Ren'Py is prior to 7.4.0, then this feature will automatically be disabled, as the ATL property 'matrixcolor' has yet to be implemented.
