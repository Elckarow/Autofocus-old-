Feature: DropShadow

Adds a drop shadow effect to the character's sprites.


allowed arguments:
xoffset              defaults to 0
yoffset              defaults to 0
offset               optional, will override both xoffset and yoffset if passed
blur                 defaults to 10
color                optional, the color of the drop shadow effect. defaults to "#000" (black)
brightness           optional, the brightness of the drop shadow effect. defaults to 1.0


If the version of Ren'Py is prior to 7.4.0, then this feature will automatically be disabled, as the ATL property 'matrixcolor' has yet to be implemented.
